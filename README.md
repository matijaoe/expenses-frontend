To run it locally:
- install pnpm
- pnpm dev (for development)
- pnpm build
- pnpm preview

Add to .env
```
VITE_API_URL=https://npjs-expenses-backend-eu.herokuapp.com/api or localhost:5000/api
VITE_CURRENCY_API_KEY=[your-api-key]
```
